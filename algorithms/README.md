# dpxcc_create_algorithms.sh

```
Usage: dpxcc_create_algorithms.sh [options]
Options:
  --algorithms-file   -a  File containing Algorithms            - Default: algorithms.csv
  --file-reference-id -f  File Reference Id name                - Default: fileid.csv
  --ignore-errors     -i  Ignore errors while adding Algorithms - Default: false
  --log-file          -o  Log file name                         - Default: Current date_time.log
  --proxy-bypass      -x  Proxy ByPass                          - Default: true
  --https-insecure    -k  Make Https Insecure                   - Default: false
  --help              -h  Show this help
Example:
dpxcc_create_algorithms.sh
```
