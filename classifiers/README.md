# dpxcc_create_classifiers.sh

```
Usage: dpxcc_create_classifiers.sh [options]
Options:
  --classifiers-file  -c  File containing Classifiers         - Default: classifiers.csv
  --ignore-errors     -i  Ignore errors while adding Classifiers - Default: false
  --log-file          -o  Log file name                         - Default: Current date_time.log
  --proxy-bypass      -x  Proxy ByPass                          - Default: true
  --https-insecure    -k  Make Https Insecure                   - Default: false
  --help              -h  Show this help
Example:
dpxcc_create_classifiers.sh
```
